# VRED_scripts
## About
This is a collection of scripts I made in preperation for the OSED exam, but can be used for other VRED activity.
## rop_finder.py
Used to pick out the most useful ROP gadgets from a text file generated by rp++.
### Usage
```
rop_finder.py [-h] [-b BAD_BYTES] [-d DLL_NAME] [-a ASLR] [-i IMAGE_BASE] [-m MAX_GADGETS] rop_output
rop_finder.py rop.txt -b "00090a0b0c0d20"
```
This script first filters out gadgets that:
- Contain bad bytes,
- Have large (>255) retns,
- Contain any calls or jumps.

It then writes all the gadgets to one file "{rop_output}-full.txt", and all the high-quality gadgets to another "{rop_output}-high.txt".
